📋 系統概述

本系統基於表面肌電圖(sEMG)技術，實現人類肌力的非侵入性計算與分析。通過實現公式1 (F = a × RMS(EMG) + b)， 為肌力評估提供科學可靠的技術方案。

✨ 核心功能

🔬 多系統支援

支援三種EMG測量系統：419-電阻式、445-耦合式、Noraxon專業設備

📊 即時分析

提供即時的時間序列分析和RMS數值計算，支援checkbox即時更新

🎯 異常值處理

自動檢測並移除異常值，提高分析結果的可靠性

📈 專業視覺化

學術級圖表呈現，符合期刊發表標準

🛠️ 本地運行

如需在本地環境運行完整的EMG分析系統：

# 克隆項目 git clone https://github.com/your-username/EMG-Analysis.git cd EMG-Analysis # 安裝依賴 pip install -r requirements.txt # 運行系統 python run.py # 訪問 http://localhost:8000/emg_report_live.html

📊 技術特點

  • 高精度時間軸: 0.1秒解析度的時間序列分析
  • 多維度比較: 支援系統間、肌肉間的靈活比較
  • 統計學驗證: 完整的統計分析和品質控制
  • 響應式設計: 適配各種螢幕尺寸和設備

🔬 理論基礎

基於Lippold (1952)和Lawrence & De Luca (1983)的研究，在等長收縮條件下， 肌電信號的RMS值與肌力輸出呈現良好的線性相關性。
